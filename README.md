## Introduction
### 이 레포지토리의 목적은 nest.js를 이용하여 각종 기능을 구현해보며 공부하는 것입니다.

## description
- 앱 생성
- board 모델 정의
- 게시판 CRUD
- 게시판 ValidationPipe
- PostgreSQL, TypeORM을 이용한 데이터베이스 연결
- 데이터베이스를 사용하여 게시판 CRUD
- 회원가입(비밀번호 bcrypt 사용)
- 로그인
- Jwt 적용
- 커스텀 데코레이터: 유저 정보 관련
- 유저별 게시글 접근 권한
- 게시글 삭제 시 soft delete 적용
